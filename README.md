
# Food Ordering System

## About
A Spring Boot application for managing food orders with MySQL database.

## 1. What is Spring Boot?
Spring Boot is a framework built on top of Spring that allows developers to create stand-alone applications quickly with minimal configuration.

## 2. What is Maven?
Maven is a build automation and dependency management tool that helps manage project libraries and build processes.

## 3. What is the purpose of pom.xml?
The pom.xml file defines the project structure, dependencies, plugins and build configuration for a Maven project.

## 4. What is the purpose of application.properties?
It stores configuration settings for the application such as database connection details, server port and other environment settings.

## 5. What does @SpringBootApplication do?
It is an annotation that combines three annotations to enable auto-configuration, component scanning and Spring configuration in one class.

## 6. Why do developers use dependency management tools like Maven?
They automatically download and manage external libraries, handle version conflicts and make it easier to share and build projects consistently.

## 7. What is a REST API?
A REST API allows applications to communicate over HTTP using methods like GET, POST, PUT and DELETE to perform operations on resources.

## 8. What is JSON?
JSON is a lightweight text format used to store and exchange data between a server and client in a readable key-value structure.

## 9. What is Dependency Injection?
Dependency Injection is when an object receives its required dependencies from an external source rather than creating them itself.

## Package Structure
- controller - Handles incoming HTTP requests
- service - Contains business logic
- repository - Manages database operations
- entity - Represents database tables
- dto - Data Transfer Objects for passing data
- config - Application configuration classes
- exception - Custom exception handling
